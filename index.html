EEG Artifact Removal using Deep Learning (Autoencoder/CNN with GUI)

Domain: Biomedical Engineering / Signal Processing / Machine Learning

Description

This project removes noise from EEG signals using a deep learning Autoencoder model. It reconstructs clean signals affected by EOG, EMG, and environmental noise.

Dataset

EEG Eye State Dataset (Kaggle) with 14 channels. Artificial noise added for simulation.

Technologies

  • Python
  • Tkinter
  • NumPy
  • Matplotlib
  • TensorFlow / Keras

Key Features

  • EEG signal upload and visualization
  • Signal preprocessing
  • Artifact simulation
  • Autoencoder-based denoising
  • Clean signal reconstruction
  • GUI comparison

Methodology

  1. Input EEG signal
  2. Normalization
  3. Noise addition
  4. Train Autoencoder
  5. Reconstruct signal
  6. Display results

Results

  • Reconstruction error ~0.01
  • Processing time < 2 seconds
  • Improved signal clarity

Conclusion

Demonstrates effective EEG signal cleaning using deep learning with a user-friendly GUI.
